Modelarea informatiilor despre pacienti pastrand datele si imaginile EEG
Aceasta platforma permite modelarea datelor pacienților și a informațiilor EEG asociate în XML în domeniul medical, cu accent pe diagnosticul și monitorizarea activităților cerebrale. Cadrul medical poate sa incarce un fisier de tip XML sau un fisier JSON pentru a prelucra datele intr-un mod mai usor. Dupa ce fisierele sunt incarcate, informatiile sunt afisate intr-un tabel
